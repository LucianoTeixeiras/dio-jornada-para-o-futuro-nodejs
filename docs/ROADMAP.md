# Roadmap - Jornada para o Futuro com Node.js

## Visão Geral

Esta trilha foi criada para guiar desenvolvedores do básico ao avançado em Node.js, com foco prático em projetos reais, boas práticas e deploy.

## Tempo Estimado Total

**~60-80 horas** (distribuídas em 4 módulos principais)

## Pré-requisitos

- Conhecimento básico de programação
- Familiaridade com JavaScript (ES6+)
- Git e GitHub básico
- Terminal/linha de comando
- Editor de código (VS Code recomendado)

## Trilha de Aprendizado

### Módulo 00: Fundamentos (15-20h)

**Objetivo:** Dominar os conceitos essenciais do Node.js e seu ecossistema.

**Tópicos:**
- O que é Node.js e como funciona
- Event Loop e assincronia
- NPM e gerenciamento de pacotes
- Módulos (CommonJS e ES Modules)
- Sistema de arquivos (fs)
- Streams e Buffers
- Debugging básico

**Entregáveis:**
- CLI tool para manipulação de arquivos
- Script de automação com npm scripts

**Tempo estimado:** 15-20 horas

---

### Módulo 01: HTTP e REST (15-20h)

**Objetivo:** Criar APIs REST robustas e escaláveis.

**Tópicos:**
- Protocolo HTTP
- Criar servidor HTTP nativo
- Express.js fundamentals
- Middlewares
- Rotas e controllers
- Validação de dados
- Tratamento de erros
- REST API design

**Entregáveis:**
- API REST completa com CRUD
- Documentação com Swagger/OpenAPI

**Tempo estimado:** 15-20 horas

---

### Módulo 02: TypeScript (15-20h)

**Objetivo:** Adicionar tipagem estática e melhorar a qualidade do código.

**Tópicos:**
- Fundamentos do TypeScript
- Tipos básicos e avançados
- Interfaces e Types
- Generics
- Decorators
- Configuração (tsconfig.json)
- Integração com Node.js e Express
- Type safety em APIs

**Entregáveis:**
- Migração da API anterior para TypeScript
- Implementação de tipos personalizados

**Tempo estimado:** 15-20 horas

---

### Módulo 03: Autenticação e Autorização (15-20h)

**Objetivo:** Implementar segurança e controle de acesso.

**Tópicos:**
- Autenticação vs Autorização
- JSON Web Tokens (JWT)
- Bcrypt e hash de senhas
- Sessions vs Tokens
- OAuth 2.0 e estratégias de login
- CORS e segurança de APIs
- Rate limiting
- Boas práticas de segurança

**Entregáveis:**
- Sistema de autenticação completo
- API protegida com diferentes níveis de acesso

**Tempo estimado:** 15-20 horas

---

## Próximos Passos (Módulos Futuros)

- **04-database:** Bancos de dados (MongoDB, PostgreSQL, ORMs)
- **05-testing:** Testes automatizados (Jest, Supertest)
- **06-docker:** Containerização e Docker
- **07-deploy:** Deploy e CI/CD
- **08-microservices:** Arquitetura de microsserviços
- **09-graphql:** GraphQL APIs
- **10-real-time:** WebSockets e aplicações em tempo real

## Como Usar Este Roadmap

1. **Leia o GUIDE.md** para entender como estudar e navegar pelo repositório
2. **Siga a ordem dos módulos** - cada um constrói sobre o anterior
3. **Complete os exercícios** - a prática é essencial
4. **Construa os projetos** - cada módulo tem um projeto incremental
5. **Use os checklists** - marque seu progresso em cada módulo
6. **Consulte o ARCHITECTURE.md** - para entender decisões de design

- ✅ Conhecimento prático de Node.js production-ready
- ✅ Experiência com TypeScript e boas práticas
- ✅ Implementação de autenticação e segurança
- ✅ Preparação para trabalhar com Node.js profissionalmente
